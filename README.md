# 智慧烹饪系统

基于 uni-app 开发的智能烹饪设备蓝牙控制APP，支持蓝牙设备扫描连接、设备操控面板、设备组管理、异常报警等功能。

## 项目概述

本项目是智慧烹饪系统的移动端APP，实现以下核心功能：

- **设备组管理**：创建设备组、跨组拖拽排序、批量开关机
- **蓝牙设备管理**：扫描、连接/断开蓝牙烹饪设备
- **设备操控面板**：开关机、12档火候调节（含快捷火候）、定时任务
- **异常报警**：接收设备异常上报（6种真实异常），持续震动提醒
- **指令日志**：记录所有收发指令和系统事件
- **测试预警**：模拟6种设备异常上报，用于测试预警功能
- **分区控制**：内外分区整体控制及分段独立控制（5段）

## 技术栈

- **框架**：uni-app（Vue 2 / Vue 3 双模式兼容）
- **蓝牙**：uni-app BLE API
- **通信协议**：12KW 电磁机芯串口明文协议（波特率9600）
- **打包平台**：Android / iOS / 微信小程序

## 快速开始

### 环境要求

- HBuilderX 3.x+
- Node.js 14+
- 支持蓝牙的 Android/iOS 设备或模拟器

### 运行

1. 使用 HBuilderX 打开项目
2. 选择运行平台（Android/iOS/微信开发者工具）
3. 点击运行

### 打包

- **Android**：发行 → 原生APP-云打包 → Android
- **iOS**：发行 → 原生APP-云打包 → iOS

## 页面说明

### 首页（index.vue）

首页是APP启动后的默认页面，以**设备组**为核心组织方式：

| 功能 | 说明 |
|------|------|
| 设备组展示 | 按用户创建的分组展示设备，默认收起 |
| 设备卡片 | 两列网格布局，显示图标、名称、火候档位、状态 |
| 创建设备组 | 右上角 [+] → 创建设备组 |
| 添加设备 | 右上角 [+] → 添加设备（蓝牙扫描或添加测试设备） |
| 编辑组名 | 点击组名直接编辑（"未分组设备"不可编辑） |
| 删除设备组 | 右上角 [+] → 删除设备组，组内设备移回"未分组设备" |
| 批量控制 | 设备组展开后显示"全部开机"/"全部关机"按钮 |
| 拖拽排序 | 长按1秒后拖拽卡片，可跨组移动设备 |
| 设备删除 | 点击卡片右上角红叉，未分组中删除，其他组中移回未分组 |
| 测试预警 | 底部"测试预警"区域，可模拟6种真实设备异常上报 |

### 控制页面（detail.vue）

控制页面是设备详情控制页，提供对单个设备的精细化控制：

| 功能 | 说明 |
|------|------|
| 开关机控制 | 开机/关机按钮，发送 >AFO/>AFF 指令 |
| 火候控制 | 加减档位调节（P0-P12），快捷火候一键切换 |
| 定时任务 | 添加倒计时（1-60分钟），支持暂停/继续/取消，定时结束自动关机 |
| 异常警报 | 显示设备异常警报，支持确认关机，持续震动提醒 |
| 指令日志 | 记录所有操作指令，按时间倒序展示 |
| 设备管理 | 重命名设备、删除设备 |
| 蓝牙消息监听 | 监听设备返回的应答和异常上报 |
| 分区控制 | 内外分区整体控制（>ACI/>ACX/>ACL）及分段独立控制（>AA1~5/>AB1~5） |

## 蓝牙通信协议

### 设备UUID

| 类型 | UUID |
|------|------|
| 服务UUID | `0000FFB0-0000-1000-8000-00805F9B34FB` |
| 写特征值 | `0000FFB2-0000-1000-8000-00805F9B34FB` |
| 通知特征值 | `0000FFB1-0000-1000-8000-00805F9B34FB` |

### 12KW 完整指令集

#### 查询类指令（手机 → 设备）

| 指令 | 说明 | 机芯上报格式 |
|------|------|-------------|
| `>AT?` | 查询三路温度 | `>A XX XX XX XX XX XX`（1路温度、2路温度、3路温度） |
| `>AP?` | 查询电参数 | `>A XX XX XX XX XX XX`（电压、功率、电流） |

#### 开关机基础控制指令

| 指令 | 功能 | 应答 |
|------|------|------|
| `>AFO` | 开机 | >OK |
| `>AFF` | 关机 | >OK |

#### 12档火力档位控制指令

| 指令 | 档位 | 应答 |
|------|------|------|
| `>AF1`~`>AF9` | 一档~九档 | >OK |
| `>AFA` | 十档 | >OK |
| `>AFB` | 十一档 | >OK |
| `>AFC` | 十二档 | >OK |

#### 内外分区整体控制指令

| 指令 | 功能 | 应答 |
|------|------|------|
| `>ACI` | 仅内区工作 | >OK |
| `>ACX` | 仅外区工作 | >OK |
| `>ACL` | 内外全区同时工作 | >OK |

#### 内区分段独立控制指令

| 指令 | 功能 | 应答 |
|------|------|------|
| `>AA1`~`>AA5` | 内区1段~5段 | >OK |

#### 外区分段独立控制指令

| 指令 | 功能 | 应答 |
|------|------|------|
| `>AB1`~`>AB5` | 外区1段~5段 | >OK |

#### 异常码（设备 → 手机主动上报）

| 指令 | 含义 |
|------|------|
| `>AE1` | 机芯超温 |
| `>AE2` | 线盘超温 |
| `>AE3` | 过流保护 |
| `>AE4` | 电流异常 |
| `>AE5` | 锅具异常 |
| `>AE6` | 线盘测温开路 |

#### 设备应答

| 应答 | 说明 |
|------|------|
| `>OK` | 设备收到指令并执行成功 |

## 测试设备（模拟模式）

点击"添加测试设备"按钮即可添加模拟设备：

- **无需真实蓝牙设备**，无需扫描
- 名称格式：`测试灶具-A-1`、`测试烤炉-B-2` 等（4种类型循环）
- 连接后可测试所有功能：开关机、调档、定时倒计时、异常报警
- 测试设备的蓝牙指令发送失败不会弹出错误提示

## 异常报警

设备上报异常时，APP会：
1. 在报警区域显示红色报警条目（异常码 + 描述 + 时间）
2. **持续震动**提示（每500ms震动一次）
3. 点击"关机"按钮发送 >AFF 关机指令并移除警报

### 模拟报警（测试用）

首页底部"测试预警"区域 → 选择设备 → 选择异常类型 → 自动跳转控制页显示警报
