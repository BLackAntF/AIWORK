## 1. 页面概述

### 1.1 页面定位

控制页面是用户点击首页设备卡片后进入的详情控制页，提供对单个设备的精细化控制。用户可在此页面开关机、调节火候档位、设置定时任务、查看异常警报和指令日志。

### 1.2 核心功能

| 功能 | 说明 |
|------|------|
| 开关机控制 | 开机/关机按钮，发送 >AFO/>AFF 指令 |
| 火候控制 | 加减档位调节（P0-P12），快捷火候一键切换 |
| 定时任务 | 添加倒计时（1-60分钟），支持暂停/继续/取消，定时结束自动关机 |
| 异常警报 | 显示设备异常警报，支持确认关机，持续震动提醒 |
| 指令日志 | 记录所有操作指令，按时间倒序展示 |
| 设备管理 | 重命名设备、删除设备 |
| 蓝牙消息监听 | 监听设备返回的应答和异常上报 |

### 1.3 页面状态

| 状态 | 说明 |
|------|------|
| 空状态 | 未选中设备，显示引导界面"请先在首页选择设备" |
| 正常状态 | 设备在线，可正常控制 |
| 离线状态 | 设备离线，控制按钮禁用，显示"设备离线"提示 |
| 警报状态 | 有活跃警报，警报区域显示红色卡片，持续震动 |

## 5. 火候控制区域

### 5.1 布局

```
┌──────────────────────────────────────┐
|  🔥 火候控制                         │
│                                      │
│     [−]        P8        [+]         │
│     ████████░░░░░░░░░░░░             │
│                                      │
│  快速火候：                           │
│  [小火] [中火] [大火] [🔥爆炒]       │
└──────────────────────────────────────┘
```

### 5.2 元素说明

| 元素 | 说明 |
|------|------|
| 区域标题 | "🔥 火候控制" |
| [−] 减档按钮 | 圆形按钮，点击减少1个档位 |
| 档位显示 | 显示当前档位 "P0" ~ "P12" |
| [+] 加档按钮 | 圆形按钮，点击增加1个档位 |
| 进度条 | 12格可视化，显示当前档位比例 |
| 小火按钮 | 一键切换至 P3 |
| 中火按钮 | 一键切换至 P6 |
| 大火按钮 | 一键切换至 P9 |
| 🔥爆炒按钮 | 一键切换至 P12 |

### 5.3 档位指令映射

| 档位 | 指令 | 说明 |
|------|------|------|
| P0 | >AFF | 关机（停火） |
| P1 | >AF1 | 一档 |
| P2 | >AF2 | 二档 |
| P3 | >AF3 | 三档（小火） |
| P4 | >AF4 | 四档 |
| P5 | >AF5 | 五档 |
| P6 | >AF6 | 六档（中火） |
| P7 | >AF7 | 七档 |
| P8 | >AF8 | 八档 |
| P9 | >AF9 | 九档（大火） |
| P10 | >AFA | 十档 |
| P11 | >AFB | 十一档 |
| P12 | >AFC | 十二档（爆炒） |

### 5.4 状态说明

#### 待机状态（P0）
```
     [−]        P0        [+]
     ░░░░░░░░░░░░░░░░░░░░
```
- 进度条全灰
- 状态文字显示"已停火"

#### 运行中（P1-P12）
```
     [−]        P8        [+]
     ████████░░░░░░░░░░░░
```
- 进度条根据档位填充
- 状态文字显示"运行中"

#### 离线状态
```
     [−]        P0        [+]    ← 按钮灰色不可点击
     ░░░░░░░░░░░░░░░░░░░░
     设备离线，无法控制
```

### 5.5 进度条颜色规则

| 档位区间 | 颜色 | 色值 |
|---------|------|------|
| P1-P4 | 蓝色 | `#2B7FFF` |
| P5-P8 | 橙色 | `#F39C12` |
| P9-P12 | 红色 | `#FF4D4F` |
| 未激活 | 灰色 | `#F5F6FA` |

### 5.6 交互说明

| 操作 | 效果 |
|------|------|
| 点击 [−] | 档位减1（最低P0），发送对应指令，记录日志 |
| 点击 [+] | 档位加1（最高P12），发送对应指令，记录日志 |
| 长按 [−] | 连续减档（每200ms减1） |
| 长按 [+] | 连续加档（每200ms加1） |
| 点击 [小火] | 一键设为P3，发送 >AF3，记录日志 |
| 点击 [中火] | 一键设为P6，发送 >AF6，记录日志 |
| 点击 [大火] | 一键设为P9，发送 >AF9，记录日志 |
| 点击 [🔥爆炒] | 一键设为P12，发送 >AFC，记录日志 |

### 7.4 异常类型（6种真实异常）

| 指令 | 异常类型 | 描述 |
|------|---------|------|
| >AE1 | 机芯超温 | 机芯温度超过安全阈值，请检查散热 |
| >AE2 | 线盘超温 | 线盘温度过高，请降低功率使用 |
| >AE3 | 过流保护 | 电流超过额定值，设备已启动过流保护 |
| >AE4 | 电流异常 | 检测到电流波形异常，请检查电源 |
| >AE5 | 锅具异常 | 未检测到锅具或锅具不匹配 |
| >AE6 | 线盘测温开路 | 线盘温度传感器开路，请检查传感器连接 |

### 7.5 交互说明

| 操作 | 效果 |
|------|------|
| 点击 [📡 模拟警报] | 弹出6种异常类型选择列表，选择后生成警报卡片，启动持续震动 |
| 点击 [关机] | 发送关机指令 >AFF，重置档位为P0，移除该条警报，记录日志 |
| 设备主动上报异常 | 蓝牙监听收到 >AE1~>AE6，自动生成警报卡片，启动持续震动 |
| 持续震动 | 每500ms震动一次，直到所有警报被确认关闭 |

| 警报关机 | `14:31:00 ⏻ 已关机：机芯超温 [>AFF]` | 确认警报时记录 |
| 温度查询 | `14:32:00 查询温度 [>AT?]` | 查询温度时记录 |
| 电参数查询 | `14:33:00 查询电参数 [>AP?]` | 查询电参数时记录 |

| 操作 | 效果 |
