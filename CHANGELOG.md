# 版本变更日志

> 智慧烹饪系统 - 版本更新记录

---

## [1.0.0] - 2024-01

### 🎉 初始版本

#### ✨ 新功能

- **蓝牙设备管理**
  - 支持 BLE 蓝牙设备扫描
  - 设备连接与断开功能
  - 设备信号强度显示
  - 自动重连机制

- **烹饪控制**
  - 5 种烹饪模式：煎炸、炖煮、蒸、烘焙、保温
  - 温度调节（0-250°C）
  - 时间设定（0-180 分钟）
  - 开始/暂停/停止控制
  - 实时温度和时间显示

- **预设菜谱**
  - 香煎牛排：200°C / 15min
  - 红烧排骨：120°C / 45min
  - 清蒸鲈鱼：100°C / 12min
  - 烤鸡翅：200°C / 25min
  - 保温米饭：60°C / 60min

- **设备管理**
  - 设备状态实时监控
  - 设备信息查看
  - 异常警报通知
  - 紧急停止功能

- **系统设置**
  - 声音提示开关
  - 自动保温设置
  - 夜间模式
  - 温度单位切换

#### 🎨 UI/UX

- 深色主题设计
- 圆形操控面板界面
- 设备状态指示灯
- 扫描动画效果
- 响应式布局适配

#### 🛠 技术栈

- uni-app 3.x (Vue 3)
- 原生 BLE API
- 跨平台支持（Android / iOS / 微信小程序）

## 版本命名规范

```
主版本号.次版本号.修订号
```

- **主版本号**：不兼容的重大变更
- **次版本号**：向下兼容的功能新增
- **修订号**：向下兼容的问题修复
