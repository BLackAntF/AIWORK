## 一、项目概述

基于 YOLOv26 的目标检测系统，结合知识库 AI 问答，支持 Web 端和手机端访问。

### 技术栈

| 层级 | 技术选型 |
|------|---------|
| 前端 | Vue 3 + Vite + Element Plus（响应式，电脑/手机通用） |
| 后端 | Python + Flask + SQLAlchemy |
| 数据库 | SQLite（开发） / MySQL（部署） |
| 检测模型 | YOLOv26（ultralytics） |
| AI 问答 | 通义千问 API + RAG 知识库 |
| 向量数据库 | ChromaDB / FAISS |

## 二、模型开发（YOLO 部分）

### 2.1 模型任务

- **任务类型**：目标检测（Object Detection）
- **模型版本**：YOLOv26（推荐 yolo26s，速度精度均衡）
- **检测类别**：待定（如病虫害种类、缺陷类型等）

### 2.2 数据准备

| 工作项 | 说明 |
|--------|------|
| 数据集收集 | 下载公开数据集 / 自行采集 |
| 数据标注 | 使用 LabelImg / LabelStudio 标注，导出 YOLO 格式 |
| 数据集划分 | 按 7:2:1 划分为训练集 / 验证集 / 测试集 |
| 数据增强 | 翻转、旋转、缩放、亮度调整等（ultralytics 内置） |

### 2.3 模型训练

```python
from ultralytics import YOLO

# 加载预训练模型
model = YOLO('yolo26s.pt')

# 训练
results = model.train(
    data='data.yaml',    # 数据集配置
    epochs=100,          # 训练轮次
    imgsz=640,           # 输入图片大小
    batch=16,            # 批次大小
    device='cuda',       # GPU / CPU
    workers=4,           # 数据加载线程
    project='runs/detect', # 输出目录
    name='exp1'          # 实验名称
)
```

### 2.4 模型评估

| 指标 | 说明 |
|------|------|
| mAP@0.5 | 平均精度，越高越好 |
| mAP@0.5:0.95 | 不同 IoU 阈值下的平均精度 |
| Precision | 精确率 |
| Recall | 召回率 |
| FPS | 推理速度 |

### 2.5 模型输出

| 输出文件 | 说明 |
|----------|------|
| `best.pt` | 验证集效果最好的权重（核心文件） |
| `last.pt` | 最后一轮训练的权重 |
| `results.csv` | 训练过程数据 |
| `results.png` | 训练曲线图 |

### 3.3 YOLO 服务封装

```python
# services/yolo_service.py
from ultralytics import YOLO
import cv2
import numpy as np
import uuid
import os

class YoloService:
    def __init__(self, model_path: str, conf_threshold: float = 0.5):
        self.model = YOLO(model_path)
        self.conf_threshold = conf_threshold
        self.class_names = self.model.names

    def detect_image(self, image_path: str, save_path: str) -> dict:
        """检测单张图片"""
        results = self.model(image_path, conf=self.conf_threshold)
        result = results[0]

        # 保存结果图
        result_img = result.plot()
        cv2.imwrite(save_path, result_img)

        # 解析检测结果
        detections = []
        class_summary = {}
        for i in range(len(result.boxes)):
            cls_id = int(result.boxes.cls[i])
            cls_name = result.names[cls_id]
            conf = float(result.boxes.conf[i])
            bbox = result.boxes.xyxy[i].tolist()

            detections.append({
                'class_id': cls_id,
                'class_name': cls_name,
                'confidence': round(conf, 4),
                'bbox': [int(x) for x in bbox]
            })

            class_summary[cls_name] = class_summary.get(cls_name, 0) + 1

        return {
            'detections': detections,
            'total_count': len(detections),
            'class_summary': class_summary
        }
```

### 3.4 AI 问答服务封装

```python
# services/llm_service.py
import chromadb
from openai import OpenAI  # 或 dashscope 等

class LLMService:
    def __init__(self, api_key: str, knowledge_dir: str):
        self.client = OpenAI(api_key=api_key, base_url='...')
        self.vector_db = chromadb.PersistentClient(path=knowledge_dir)
        self.collection = self.vector_db.get_collection('knowledge')

    def retrieve_knowledge(self, query: str, top_k: int = 3) -> list:
        """从向量库检索相关知识"""
        results = self.collection.query(query_texts=[query], n_results=top_k)
        return results['documents'][0]

    def ask(self, question: str, context: str = '') -> dict:
        """回答问题"""
        # 检索相关知识
        knowledge = self.retrieve_knowledge(question)
        knowledge_text = '\n'.join(knowledge)

        # 构建提示词
        prompt = f"""
        参考以下知识库内容回答用户问题：
        {knowledge_text}

        检测上下文：{context}

        用户问题：{question}

        请用中文回答，回答要准确、专业。
        """

        # 调用大模型
        response = self.client.chat.completions.create(
            model='qwen-turbo',
            messages=[{'role': 'user', 'content': prompt}]
        )

        return {
            'answer': response.choices[0].message.content,
            'sources': knowledge
        }
```
